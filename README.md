# 📝 Notes App (Python Tkinter)

This is a simple Notes application made with Python and Tkinter.

## 🎯 About the project

This project allows users to write and save simple notes using a graphical interface.  
It is a beginner-friendly project to practice Python GUI development.

## ⚙️ Features

- Add notes using input field
- Display notes in a list
- Simple and clean GUI (Tkinter)
- Easy-to-use interface

## 🚀 How it works

- Type a note in the input field
- Click the "Add" button
- The note appears in the list
- Input field is cleared automatically

## ▶️ How to run

1. Install Python 3
2. Run the file:

```bash
python notes_app.py
